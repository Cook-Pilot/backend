# 04. CookPilot 화면 구조

작성 기준: 2026-07-10 확정 MVP

## 1. 화면 목록

| 화면 ID | 화면명 | MVP 여부 | 목적 |
|---|---|---|---|
| S-01 | 레시피 선택 | P0 | 조리할 레시피를 고른다. |
| S-02 | 조리 시작/버전 확인 | P0 | 기본 버전 또는 내 버전으로 시작한다. |
| S-03 | 조리 화면 | P0 | 현재 단계, 타이머, 음성 모드를 제공한다. |
| S-04 | 조리 후 피드백 | P0 | 결과 만족도와 코멘트를 입력한다. |
| S-05 | 개인 레시피 버전 | P0 | 생성된 내 버전을 확인한다. |

## 2. 사용자 흐름

```text
S-01 레시피 선택
-> S-02 조리 시작/버전 확인
-> S-03 조리 화면
-> S-04 조리 후 피드백
-> S-05 개인 레시피 버전
-> S-01 같은 레시피 재진입 시 내 버전 제안
```

## 3. S-01 레시피 선택

### 목적

사용자가 MVP용 제한 레시피 중 조리할 메뉴를 선택한다.

### 구성

- 앱 이름
- MVP 레시피 목록
- 최근 조리한 레시피
- 내 버전 배지
- 레시피 카드

### 레시피 카드

```text
[라면]
내 버전 있음
마지막 조리: 2026-07-10
주요 수정: 스프 15% 감소, 면 30초 짧게
[조리 시작]
```

## 4. S-02 조리 시작/버전 확인

### 목적

이번 조리에 사용할 레시피 버전을 확인하고 조리 세션을 시작한다.

### 구성

- 레시피명
- 기본 버전/내 버전 표시
- 이번 조리에 적용될 주요 조정값
- 조리 시작 버튼

### MVP 원칙

- 복잡한 조리 전 설정은 MVP에서 최소화한다.
- 개인 버전이 있으면 내 버전을 우선 제안한다.
- 사용자는 기본 버전으로 되돌릴 수 있다.

## 5. S-03 조리 화면

### 목적

조리 중 사용자가 현재 해야 할 일, 남은 시간, 음성 상태를 즉시 확인하고 음성으로 명령할 수 있게 한다.

### 필수 구성

```text
[레시피명 / 단계 번호]
[현재 단계 조리법]
[남은 시간 또는 진행 상태]
[음성 상태: 듣는 중 / 처리 중 / 다시 말해주세요]
[빠른 조작: 이전 / 다시 듣기 / 1분 추가 / 다음]
[조리 완료 / 중단]
```

### 화면 원칙

| 원칙 | 설명 |
|---|---|
| 현재 단계 우선 | 현재 조리법이 화면에서 가장 잘 보여야 한다. |
| 타이머 명확성 | 남은 시간이 크게 보여야 한다. |
| 음성 상태 노출 | 앱이 듣고 있는지 사용자가 알아야 한다. |
| 버튼 fallback | STT 실패 시에도 동일 조작이 가능해야 한다. |
| 화면 이탈 제어 | 화면 이탈 시 음성 수신이 종료된다는 상태가 명확해야 한다. |

### 상태 예시

```text
라면 1/3

물 500ml를 넣고 끓이세요.

02:14

듣는 중

[이전] [다시 듣기] [1분 추가] [다음]
```

## 6. S-04 조리 후 피드백

### 목적

조리 결과를 개인 레시피 개선 데이터로 전환한다.

### 구성

- 만족도
- 자유 코멘트
- 다음에 바꿀 점
- 저장 버튼

### 입력 예시

```text
만족도: 3/5
코멘트: 너무 짰다. 면이 조금 퍼졌다.
다음에: 스프를 덜 넣고 면을 덜 끓이고 싶다.
```

## 7. S-05 개인 레시피 버전

### 목적

조리 후 피드백이 어떻게 내 버전으로 반영되었는지 보여준다.

### 구성

- 레시피명
- 내 버전 요약
- 변경된 단계/재료/시간
- 반영 근거
- 다음에 이 버전으로 조리하기 버튼

### 예시

```text
라면 - 내 버전

- 분말스프: 1봉 -> 0.85봉
- 면 끓이는 시간: 3분 -> 2분 30초

근거:
- 지난 조리 피드백: "너무 짰다", "면이 퍼졌다"
```

## 8. 예외 상태

| 상태 | 화면 대응 |
|---|---|
| 마이크 권한 없음 | 음성 모드 비활성, 버튼 fallback 안내 |
| STT 인식 실패 | "다시 말해주세요" 표시 |
| LLM 응답 지연 | "확인 중" 표시, 로컬 타이머는 계속 진행 |
| 네트워크 오류 | 로컬 명령만 가능, 예외 질문은 나중에 다시 시도 |
| 조리 화면 이탈 | 음성 수신 종료, 세션 복귀 버튼 표시 |
